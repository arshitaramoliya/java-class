# Java — Number Sorting Logic

A focused Java program demonstrating conditional-based sorting of three numbers in descending order — without using built-in sort methods.

## What It Does

`Test11.java` takes three integer values and determines their descending order using nested `if-else` comparisons, printing the sorted result.

## Concepts Covered

- Nested conditional logic
- Manual comparison-based sorting
- Understanding sort algorithms at the logic level before using library methods
